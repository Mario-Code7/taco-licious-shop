# Taco-licious App

This is a **Taco shop Application** built in Java that helps you **navigate through menus to add your favorite foods**. All receipts are saved to a `.txt` file and can be filtered/viewed in different ways.

---

## Files Included
- `Program.java`: The main Java program that runs the taco shop.
- `.txt`: The file where all receipts are saved.

---

## Features
- Add taco, drink, chips/salsa, and signature tacos.

---

## Taco Menu
- (1) New Order 
- (0) Exit

## Food Menu
- (1) Add Taco
- (2) Add Drink
- (3) Add Chips & Salsa
- (4) Add Signature Taco
- (5) Checkout
- (0) Cancel Order
